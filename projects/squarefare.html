SquareFare

Software Engineer Intern

Overview

Built a barcode driven prep and plating system that converts planned meals into scannable stickers and live ingredient breakdowns per station. A cook scans a label and instantly sees exact prep quantities for that meal, replacing paper sheets and enabling smooth scaling during busy weeks.

Also developed a customer website where users select meals a la carte or via subscriptions. Selections sync to a central spreadsheet and production view, removing manual data entry and reducing order errors. Together, these tools turned a manual process into a reliable, trackable workflow that supports growth.

Role and Stack

  • Role: Software Engineer Intern
  • Focus: barcode generation, station logic, real time prep quantities, order intake automation, error reduction
  • Stack: Python, Flask, JavaScript, React, Google Sheets API, SQLite, QR and barcode libraries
  • Ops: structured data model for meals, components, and stations; scan flow and audit trail

Key Features

  • Sticker and barcode generator for each planned meal with order and component metadata.
  • Scan to prep view that shows exact ingredient quantities for the current station and meal.
  • Kitchen dashboard with live counts, flags for low inventory, and error checks.
  • Customer site for a la carte and subscriptions that syncs selections directly into production.
  • Data validation and reconciliation to prevent duplicates and catch mismatches early.
