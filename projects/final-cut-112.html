Final Cut 112

Python Video Editor · CMU Graphics

Overview

Final Cut 112 is a Python-based video editing tool built with the CMU Graphics framework. It supports filters, stitching, cutting, resizing, overlays, and transitions, with a functional timeline UI and multiple panes for media, filters, and transitions. Basic controls like opacity and hue are exposed for quick adjustments, and simple transitions like fades are built in.

The goal was to create a lightweight, accessible editor that demonstrates core NLE concepts end to end: importing media, arranging clips on a timeline, applying layered effects, rendering previews, and exporting final cuts.

Role and Stack

  • Role: Designer and sole developer
  • Stack: Python, CMU Graphics, NumPy, basic image/video IO libraries
  • Focus: timeline model, effects pipeline, UI state management, preview rendering

Key Features

  • Functional timeline with clip selection, trimming, and drag-to-reorder.
  • Filters and adjustments: opacity, hue, brightness, and stackable effects.
  • Transitions: fades and simple cross-dissolves between adjacent clips.
  • Overlays: text and image layers with position and blend control.
  • Clip tools: stitch, cut, and resize with visual feedback.
  • Multi-pane UI: media browser, filters panel, transitions panel.
